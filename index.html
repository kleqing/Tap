Support

How to play?

Just press space and jump. Avoid every obstacle in the way and get close to the destination!


Which device can I use to play this game?

This game is build native with unity for Windows. Don't worry, there will be a version that support for Linux and macOS soon!


Is Tap open source?

Tap is fully open source. Details on how you can contribute to the Sileo project can be found in the readme!


Does I need to install .NET before play?

This game is build from .NET 8. Make sure you have .NET runtime 8 preinstalled! Otherwise, you can download here.
